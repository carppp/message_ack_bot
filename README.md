# Discord Message Ack Bot

一個具備自動回應與關鍵字計數器功能的 Discord 機器人。

## 功能介紹

### 1. 自動回應 (全域功能)
這組指令在所有機器人加入的伺服器中通用的。

*   **儲存回應**：`..[關鍵字] [內容]`
    *   範例：`..你好 嗨！你好呀`
    *   若同個關鍵字儲存多次，觸發時會隨機挑選一個回覆。
*   **觸發回應**：`...[關鍵字]`
    *   範例：`...你好` -> 機器人回覆：`嗨！你好呀`
*   **刪除回應**：`..del [關鍵字]`
    *   範例：`..del 你好`

---

### 2. 關鍵字計數器 (伺服器獨立功能)
這組指令會根據不同的伺服器 (Guild) 獨立計算次數，互不干擾。

*   **新增計數器**：`..add count [偵測關鍵字] [偵測後回話內容] [統計顯示名稱]`
    *   **支援變數**：在「回話內容」中使用 `{count}` 會自動替換為該使用者的目前累計次數。
    *   範例：`..add count 笨蛋 你已經說了{count}次笨蛋了喔 笨蛋榜`
*   **自動偵測**：
    *   當一般訊息中包含「偵測關鍵字」時，機器人會自動為該發言者增加 1 次計數。
    *   若有設定「回話內容」，機器人會自動發送該訊息。
*   **查詢統計榜**：`...[統計顯示名稱]`
    *   範例：`...笨蛋榜`
    *   顯示該伺服器內所有人的計數排名（由高到低）。
*   **刪除計數器**：`..del count [偵測關鍵字]`
    *   刪除該伺服器內該關鍵字的所有計數資料。

---

## 技術說明
*   **開發語言**：Node.js
*   **主要套件**：`discord.js`, `dotenv`
*   **資料儲存**：`memory.json` (自動生成，包含回應與計數器資料)

## 安裝與執行
1.  複製 `.env.example` 並更名為 `.env`，填入你的 `DISCORD_TOKEN`。
2.  執行 `npm install` 安裝套件。
3.  執行 `node bot.js` 啟動機器人。
